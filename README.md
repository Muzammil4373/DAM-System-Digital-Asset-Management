# DAM System — Digital Asset Management



## Quick Start

### Backend
```bash
cd backend
npm install
cp .env   
npm run dev            # http://localhost:5000
```

### Frontend
```bash
cd frontend
npm install
npm start              # http://localhost:3000
```

## Features
- ✅ Upload images, PDFs, videos (drag-and-drop or click)
- ✅ Real-time upload progress bar
- ✅ Gallery view (grid + list toggle)
- ✅ Search by filename or tag
- ✅ Filter by file type and date range
- ✅ In-browser preview (images, video, PDF)
- ✅ Download assets
- ✅ Tag management (inline editing)
- ✅ Delete assets (file + DB record)
- ✅ Error handling for unsupported types and large files

## Tech Stack
- **Frontend**: React 18, Axios, custom CSS
- **Backend**: Node.js, Express, Multer, UUID
- **Database**: MongoDB with Mongoose
